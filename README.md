# AI Cover Letter Generator

A simple frontend project that generates a professional cover letter based on user inputs like candidate name, job role, company name, and key skills.

The goal of this project was to focus on clean UI design, form handling, dynamic content generation, and user interaction using HTML, CSS, and JavaScript.

## Features

- Professional and responsive UI
- Dynamic cover letter generation
- Input validation
- Copy to Clipboard functionality
- Smooth user experience with loading interaction

## Technologies Used

- HTML
- CSS
- JavaScript

## How It Works

Users enter their details into the form, and the application generates a customized cover letter using a predefined template string.

The generated output appears inside a dedicated copy box for better readability and user experience.
